# Westbrook Lab — Cancer Transcriptomics Analysis

Jupyter notebooks containing samples of data analysis and visualizations behind figures from my PhD thesis at Baylor College of Medicine (Westbrook Lab).

## Research Context

This work investigates RNA splicing dysregulation in cancer, using transcriptomic data to identify splicing patterns associated with tumor biology. The analyses span differential splicing analysis, gene expression profiling, and data visualization for publication-quality figures.
