# Timeline Plot — Desktop Gantt Chart Tool

A desktop application for building Gantt charts locally, designed for individuals who need timeline figures without uploading sensitive data to cloud services.

## Why This Exists

This is a straightforward tool to create Gantt charts for those without experience in using existing software tools. The design choice for creating this into a desktop app is to allow local usage which maintains confidentiality of data and information. 

## Features

- Build and edit Gantt charts through a simple desktop GUI
- Import an excel file and export a png 
- No Python installation necessary
- Lightweight and fast

## Installation & Usage

### Option 1
Download repository and double click make_timeline_plot.exe (For windows systems)

### Option 2
```bash
git clone https://github.com/hobrian/Work_samples.git
cd make_timeline_plot
pip install -r requirements.txt
python make_timeline_plot.py   
```
